Steps to run application:
1- Go to the respective directory in command promt and run the below commands
2- npm install
3- node app.js
4- open http://localhost:3000/ in your browser

Here I have used NoSql (Mongodb atlas as database).
The structure of many different forms of data is more easily handled with a NoSQL database, so I used Mongodb.

Operations with can be performed on this application:
For Student:
1- Add Student data

For Admin:
1- Add Student data
2- view All student data
3- Update student data
4- Delete student data
